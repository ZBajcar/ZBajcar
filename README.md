## ‚ö° Burrow

[Burrow](https://burrow-io.github.io/) is a self-hosted data ingestion pipeline and retrieval API that streamlines and accelerates the development of RAG applications. It transforms unstructured documents into searchable vector embeddings with advanced parsing, and allows retrieval with techniques such as hybrid search, metadata filtering and reranking.

**Technologies Used:** Python, TypeScript, JavaScript, Terraform, AWS (ECS, Lambda, ALB, EventBridge, SQS, S3, RDS, DynamoDB, CloudFront)
